# Mode: training - Learning Investment Evaluation

For each course/certification, evaluate 6 dimensions:

| Dimension | What it measures |
|----------|-------------------|
| North Star alignment | Does it move the candidate toward target roles? |
| Recruiter signal | How does it read on a resume? |
| Time and effort | Weeks x hours/week |
| Opportunity cost | What high-value work gets displaced? |
| Risk | Outdated content, weak brand, too basic |
| Portfolio artifact | Does it produce demonstrable output? |

## Verdicts

- `DO` -> 4-12 week plan with weekly deliverables.
- `DO NOT DO` -> better alternative with rationale.
- `TIMEBOXED DO` -> max X weeks, essentials only.
