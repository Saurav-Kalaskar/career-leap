# Mode: project - Portfolio Project Evaluation

Score each candidate project across 6 dimensions (1-5):

| Dimension | Weight | 5 = ... | 1 = ... |
|----------|--------|---------|---------|
| Signal for target roles | 25% | Directly proves target skill | Unrelated |
| Uniqueness | 20% | Distinctive | Commodity |
| Demo-ability | 20% | 2-minute live demo possible | Code-only/no clear demo |
| Metric potential | 15% | Clear measurable outcomes | No measurable outcomes |
| Time to MVP | 10% | <=1 week | 3+ months |
| STAR potential | 10% | Rich trade-off story | Implementation only |

## Interview Pack Requirements

For approved projects:
1. One-pager: product, architecture, metrics, evaluation plan.
2. Demo: live URL or 2-minute walkthrough.
3. Postmortem: what worked, what failed, and mitigations.

## 80/20 Plan

- Week 1: MVP + core metric.
- Week 2: polish + interview pack.

## Verdicts

- `BUILD`
- `SKIP`
- `PIVOT TO [alternative]`
